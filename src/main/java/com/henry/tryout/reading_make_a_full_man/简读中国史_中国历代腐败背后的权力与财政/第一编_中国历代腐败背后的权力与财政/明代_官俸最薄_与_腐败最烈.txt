明代俸禄的确是非常微薄，这有两方面原因：
一是 受到了元代薄俸制度的影响，
另一个，也是最根本的原因，则是 明太祖朱元璋的统治思路。

因为出身贫苦，朱元璋终生 痛恨贪官污吏。
...

朱元璋不了解，正是 元代的薄俸制度 助长了 官吏的贪污腐败。
朴素的农民思维 让他认为 当官的没有几个好人。
因此，在中国历朝历代中，朱元璋 对官员们 是最苛刻的。

在制定俸禄制度的时候，朱元璋 对自己的亲戚家人 待之唯恐不厚，
宗室贵族的俸禄 异常丰厚。
皇子封赏为 亲王之后，年俸禄有一万石，是最高官员的 7倍之多，
这还不包括 丰厚的土地赏赐。
但同时，朱元璋 又以 开国之初，民生凋敝，“百姓财力俱困，如初飞之鸟” 为由，
把官吏的俸禄 定得出奇地低，对于普通官员 以薄俸为主流。

明代俸禄之薄 体现在以下四个方面：
#1 明代官员 没有了职田。
从唐朝开始，地方现任官员俸禄中 都会有一项 特别稳定的收入————职田收入。
但朱元璋 废除了这个制度，把丰腴之地 都封赏给了 自己的子孙。

#2 禄米的数量也大大下降。
...
明代官员的收入，不过是 唐代的十分之二到十分之三。
...
所以，明代低薪制度 贯穿了 上中下级 所有的官员。

#3 不仅如此，朱元璋 甚至舍不得 给退休官员 开工资；

#4 除了低薪制度外，朱元璋还取消了 官员的许多其他特权。
比如，唐宋两代，官员的子弟 会得到“荫封”，可以直接做官。
但是朱元璋 取消了这种做法。
虽然大官的子弟 仍旧会得到“荫叙”，但所叙的 只是禄米，而不是官职。
想要做官的话，仍旧需要参加考试。

另外，在明朝以前，官场上一直实行“以官抵罪”。
就是说 如果官员犯罪，以降职或者夺官 作为一种抵罪措施。
但是朱元璋 也取消了这种特权。
官员犯罪，与百姓一样，该坐牢的坐牢，该流放的流放，不会有一点的宽容。

很明显，在分配帝国利益蛋糕的时候，朱元璋 把官僚体系排除在外了。

...

这样的低薪不足以资生，导致很多官员难以养家糊口。

===
朱元璋时代，俸禄已经如此之低。
按理说，在朱元璋去世之后，随着经济恢复，社会发展，明代的统治者们 应该大幅度提高俸禄才对。
但实际上，洪武朝之后，明代的俸禄水平 非但没有提高，反而 逐步降低。

主要原因是 明代俸禄的“折色”制度。

明代的俸禄主要是 使用粮食来发放的。
但是 当国库中的粮食不够用的时候，偶尔也会发点其他的东西来代替，这就是所谓的 折色。
折色 最高始于洪武年，但之后就越来越 普遍。

建文四年，户部称...，也就说，因为国家粮食储备不足，
所以各处 都司发工资的时候，一半发钞票，一半发粮食。
...
折色渐渐成为定制。

问题和元代一样,明代的钞票经常 严重贬值。
...

如果按照半粮半钞来算的话，官员实际 只能够领到部分粮食，
以钞票折算的部分 基本就算是 给国家做贡献了，工资相当于 直接降低了百分之五十。
明代的皇帝们 就是这样 和官员们 耍流氓的。

时间长了，百官当然纷纷抗议，皇帝们于是不发钞票，改发其他东西。
发胡椒和苏木，胡椒是调料，而苏木是燃料。

为什么发这些东西呢?
因为这些是 从国外进口的奢侈品，郑和下西洋的时候 进口多了，宫中堆积多年，
眼瞅着就要放坏了，所以拿出来 给百官充工资用。

...
反正宫中什么东西多 就发给百官什么。

为什么要用这些东西 来 给百官发工资呢？
当然是因为 国库没钱了。
为什么大明朝会穷成这样呢？
主要原因是，天下的银子 都发给朱元璋的后代了。

朱元璋规定，他的后代 由于身份高贵，不能从事任何社会职业，
增加收入的方式只有一个渠道，那就是多生孩子。多生一个孩子，国家 就按照等级 多发放一份俸禄。

这样，国际财政 就出现了 难以支撑的窘况。
嘉靖十四年，朝廷曾经算过一笔账，
天下岁供 京师 粮食 四百万石，
而 诸王府的禄米支出 竟然达到了 八百五十三万石，缺口是  四百五十三万石??

好比原本这东西就只值10块钱，皇帝却非要说 能够值100块钱。
...
明代皇帝们的无赖嘴脸 就是这样的毫无遮拦。

...
朝廷发俸禄时，在银子、粮食、布匹之间反复折腾，但是不管怎么折腾，
反正是 越来越有利于皇帝，不利于百官。

===
明代官场贪腐发展曲线图，与俸禄高低 有着直接的关系。
明代贪腐之风盛行，恰恰发生在 洪武朝之后，与俸禄折色制度同步。
折色越低，贪污腐化发展得就越厉害。

===
处理官员贪腐程度 登峰造极，明代 也是中国历史上 吏员腐败最严重的时期之一。
原因也和低薪制度 密切相关。

在规定官员低薪制的同时，明代也规定了吏员的低薪制。

在经济收入低下的同时，明代吏员的社会地位 比起宋代 又进一步降低。
明太祖和明成祖 都颁布上谕说，因为 吏员心术已坏，不准进行科举，
对他们进行 毫不掩饰的歧视。

吏员待遇和地位如此之低，但是到了明代中后期，社会上有一个奇怪的现象，
很多老百姓拼了命也要当吏员。
因为吏员的实际收入 其实很高。

明代吏员们的薪水虽然几乎等于没有，但是额外收入却林林总总。
第一项额外收入叫做“顶首银”，就是 一个吏员在升走调用 或者 退休的时候，会推荐一个人来接班。
被推荐的人 就要给他一笔钱 作为酬谢。

...
明代各部院的吏员 大都来自于 江浙一带，他们最初是
随着 永乐皇帝迁都北京 而 入驻京师的，后来这份工作 就时代相袭了下来。

明代大儒 黄宗羲说，当时 虽然没有 世袭之官，但是却有 世袭之吏，
他们一代又一代 泡在公文之中，精熟 各种官司 典章 掌故，
所以 在官长 要求参考以前的案例时，他们可以随手拿出 符合自己营私需要的案例。


因为缺少专业知识，明代那些掌握了“大经大法”的六部官员 很容易被书吏所把持。
在各部之中，实际主持事务的，往往都是 积年老吏，
他们经手处理的公文 很少被那些 弄不清楚“成例”的 主管官员 否定。

...

吏员们的收入当然不止 顶首 这一项。
海瑞记载，淳安县的吏员们 还有很多 常例陋规收入。
...

这些常例陋规公然记载，并且有定数，这是因为
地方吏员几乎没有工资，这些常例 就相当于 地方政府 在摊牌提留中，为吏员们安排的半合法收入。
除此之外，吏员在地方事务中 拥有诸多广泛的权力，他们 营私的机会还有很多。

==
户房 管理国家钱粮，营私肥己的机会尤其多。
增派、侵匿、挪用...

==
礼房表面上是清水衙门，但是吏员们 也是敛财有术。
比如在采办 考试、祭祀、旌表(牌坊或匾额)和乡饮大典等相关物资时，会 以次充好，赚取差价。
科举考试时...
祭祀时...
旌表时...
乡饮大典时...
县考时，帮考生作弊；
办理“下程酒席”时，勒索里甲。

==
县级政府兵房的主要职能之一 是征兵。
由于明代军人待遇恶劣，百姓视之为苦役，想方设法逃避，所以 兵房吏员 有很多机会 可以从征兵中牟利。

==
刑房吏员 负责 协理刑名，处理案件，因此 其营私手法百姓更为熟悉。
打官司时，当事人必须贿赂 刑房吏员，才有获胜可能。

==
工房吏员 负责主持地方上的各种工程建设，因此有很多机会可以 勒索民户。

总而言之，明代吏员 营私枉法已经非常普遍。
事实上，大明王朝后期的腐败 已经到了 沦肌浃(jia)骨的地步，
侵蚀到了 社会肌体的每一个细胞。

办一件事需要多少贿赂，明码标价，
“权门之厉害如响，富室之贿赂通神。
钝口夺于佞(ning)词，人命轻于酷吏”

小小的衙门胥吏 通过把持官府可以 家资上万。
